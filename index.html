What This Is

A living catalog of website demos

This repo is a centralized portfolio and demo library for websites, landing pages, and concept builds created for different businesses. The system is built for rapid prototyping — build fast, store clean, showcase better.

Each demo is a real, browsable website with full design, content, and structure. New demos get added as they're built.

  • Build a website demo with AI tools — drop it into the repo as a new project folder

  • Each demo becomes a portfolio piece instantly, organized by industry and business type

  • Show clients real, live examples that match their industry — not generic templates

  • The homepage auto-organizes everything into one clean, professional showcase
